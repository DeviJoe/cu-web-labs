# SSTI CTF Challenge

Веб-приложение для студенческого CTF с уязвимостью Server-Side Template Injection.

## Описание

Student Portal - простое веб-приложение для студентов с функциями поиска, авторизации и информационными страницами.

## Требования

- Docker
- Docker Compose

## Развертывание

### Запуск сервиса

```bash
docker-compose up -d
```

### Остановка сервиса

```bash
docker-compose down
```

### Проверка логов

```bash
docker-compose logs -f
```

## Доступ к сервису

После запуска сервис будет доступен по адресу: `http://localhost:5000`

## Структура приложения

- `/` - Главная страница
- `/login` - Страница авторизации
- `/search` - Поиск
- `/about` - Информация о приложении

## Цель

Найти и получить флаг формата `centralctf{...}`
