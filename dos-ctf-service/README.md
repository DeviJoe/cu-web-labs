# DOS CTF Challenge Service

Сервис валидации email-адресов для студенческого CTF соревнования.

## Описание

Это веб-приложение предоставляет простой интерфейс для проверки корректности email-адресов. Сервис анализирует введенные данные и сообщает, соответствует ли email правильному формату.

## Требования

- Docker
- Docker Compose

## Установка и запуск

1. Клонируйте репозиторий или скопируйте файлы проекта

2. Перейдите в директорию проекта:
```bash
cd dos-ctf-service
```

3. Запустите сервис с помощью Docker Compose:
```bash
docker-compose up -d
```

4. Сервис будет доступен по адресу: http://localhost:5000

## Остановка сервиса

```bash
docker-compose down
```

## Использование

1. Откройте браузер и перейдите на http://localhost:5000
2. Введите email-адрес в форму
3. Нажмите кнопку "Validate Email"
4. Получите результат валидации

## Цель задания

Найдите флаг формата `centralctf{}`
